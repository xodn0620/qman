■ 실행 방법
  • 이 폴더 안의 「QMan.exe」를 더블클릭하면 됩니다.
  • Windows PC(64비트)에서 사용하세요.


■ 라이선스·고지 (배포 시)
  • 같은 폴더에 LICENSE, THIRD_PARTY_NOTICES.txt 가 있으면 함께 넣어 주세요.
  • 회사 내부 배포·감사 제출 시에도 위 파일을 빼지 말고 같이 제공하는 것이 좋습니다.


■ 내 데이터(매뉴얼·설정)는 어디에 저장되나요?
  • 문서와 설정은 항상 「QMan.exe가 있는 폴더」 아래 data\qman.db 한 곳에만 저장됩니다.
  • 처음 실행하면 data 폴더가 없으면 자동으로 만들어집니다.
  • 다른 드라이브나 다른 경로의 DB 파일은 사용하지 않습니다. exe와 같은 폴더를 통째로 백업하면 됩니다.


■ 검색이 조금 느리거나, “벡터” 관련 안내가 나올 때
  • 첫 실행 시 검색 기능을 위해 QMan.exe 옆에 native 폴더가 생길 수 있습니다. 정상 동작입니다.
  • 이 배포본에 검색용 확장이 포함되어 있지 않으면 앱은 그대로 쓸 수 있고, 검색은 다른 방식(코사인 등)으로 동작합니다.


■ 질문했는데 답에 “참조 문서”가 안 나오거나 엉뚱할 때
  • 채팅 화면 왼쪽에서 고른 카테고리와, 「매뉴얼 관리」에서 문서를 올린 카테고리가 같은지 확인해 보세요.
  • 설정에서 임베딩 모델을 바꾼 뒤에는 기존 문서를 다시 업로드해서 인덱싱해야 할 수 있습니다.


■ 폴더가 생겨 보이면
  • QMan.exe 옆에 data\, native\ 가 생길 수 있습니다. 모두 프로그램이 만드는 폴더입니다.
